# DebateWithMe - Real-Time Multiplayer Debate Platform

A modern web application for practicing and competing in debate formats with real-time multiplayer functionality.

## Features

- **Multiple Debate Styles**: World Schools, British Parliamentary, and Quickfire Clash
- **Real-Time Multiplayer**: Global matchmaking with live transcript sharing
- **AI-Powered Feedback**: Detailed analysis of speech content, delivery, and strategy
- **Speech Recognition**: Real-time transcription during debates
- **Professional UI**: Modern, responsive design with seamless user experience

## Getting Started

This is a Next.js project bootstrapped with `create-next-app`.

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start.

### Production

```bash
npm run build
npm start
```

## Multiplayer Features

- **Global Lobby System**: Players from around the world can join debates
- **Real-Time Updates**: Live transcript sharing between participants
- **Turn-Based Speaking**: Structured debate rounds with synchronized timing
- **Cross-Platform**: Works on all devices and browsers

## Debate Formats

1. **World Schools Debate**: 6-person team format with structured speeches
2. **British Parliamentary**: 8-person debate with multiple roles
3. **Quickfire Clash**: Fast-paced 2-person debate with 5 rounds

## Technology Stack

- **Frontend**: Next.js 16, React 19, TypeScript
- **Backend**: Next.js API Routes
- **Styling**: Tailwind CSS
- **Real-Time**: Server-based multiplayer infrastructure

## Deployment

Deployed on Vercel with automatic CI/CD from GitHub.
